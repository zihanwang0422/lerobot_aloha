# 双臂主从遥操作 + 数据采集 + 训练 ACT (Aloha)

## 整体架构

```
主臂(Leader) × 2  →  bi_so_leader（遥操作器）
                         ↓ 关节角度
从臂(Follower) × 2 ←  bi_so_follower（机器人）
                         ↓ 录制
                     LeRobotDataset（数据集）
                         ↓ 训练
                     ACT Policy（双臂 Aloha）
```

动作维度：双臂共 **14 维** = 左臂 6 关节 + 1 夹爪 + 右臂 6 关节 + 1 夹爪

---

## 第一步：串口权限 & 查找端口

```bash
sudo usermod -aG dialout $USER   # 重新登录后生效
sudo chmod 666 /dev/ttyUSB*

# 查找各臂串口
lerobot-find-port

# 查找摄像头索引
lerobot-find-cameras
```

```bash

lerobot-calibrate \

lerobot-calibrate \

# 右主臂
lerobot-calibrate \
    --teleop.type=so_leader \
    --teleop.port=/dev/ttyUSB3 \

```

## 第四步：采集数据集

```bash
export HF_USER=你的HuggingFace用户名

lerobot-record \
  --robot.type=bi_so_follower \
  --robot.id=bimanual_follower \
  --robot.left_arm_config.port=/dev/ttyUSB0 \
  --robot.right_arm_config.port=/dev/ttyUSB1 \
  --robot.left_arm_config.cameras='{
    wrist_left: {"type": "opencv", "index_or_path": 0, "width": 640, "height": 480, "fps": 30}
  }' \
  --robot.right_arm_config.cameras='{
    wrist_right: {"type": "opencv", "index_or_path": 1, "width": 640, "height": 480, "fps": 30},
    top: {"type": "opencv", "index_or_path": 2, "width": 640, "height": 480, "fps": 30}
  }' \
  --teleop.type=bi_so_leader \
  --teleop.id=bimanual_leader \
  --teleop.left_arm_config.port=/dev/ttyUSB2 \
  --teleop.right_arm_config.port=/dev/ttyUSB3 \
  --dataset.repo_id=${HF_USER}/bimanual-aloha-task \
  --dataset.num_episodes=50 \
  --dataset.single_task="双臂协作——将红色方块从左臂传递到右臂" \
  --display_data=true
```

录制按键：
- `Space` — 开始/暂停当前 episode
- `Right` — 保存并开始下一个 episode
- `Esc` — 保存并退出

建议采集量：简单任务 ≥ 50 条，复杂任务 ≥ 200 条。

---

## 第五步：训练 ACT 双臂策略

```bash
lerobot-train \
  --dataset.repo_id=${HF_USER}/bimanual-aloha-task \
  --policy.type=act \
  --policy.chunk_size=100 \
  --policy.n_action_steps=100 \
  --policy.input_features='{
    "observation.state":              {"type": "STATE",  "shape": [14]},
    "observation.images.wrist_left":  {"type": "VISUAL", "shape": [3, 480, 640]},
    "observation.images.wrist_right": {"type": "VISUAL", "shape": [3, 480, 640]},
    "observation.images.top":         {"type": "VISUAL", "shape": [3, 480, 640]}
  }' \
  --policy.output_features='{"action": {"type": "ACTION", "shape": [14]}}' \
  --training.num_steps=100000 \
  --training.batch_size=8 \
  --output_dir=outputs/train/bimanual_act \
  --wandb.enable=false
```
